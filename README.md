# Arrow Flow 🎮

A complete Flutter puzzle game — directional arrows, multiple grid shapes, 5 visual themes, and 200+ levels.

## Project Structure

This project is built section by section:

- **Section 1** — Project Skeleton, pubspec.yaml, Design System, Core Architecture
- **Section 2** — Design System (Colors, Typography, Themes)
- **Section 3** — Splash Screen
- **Section 4** — Onboarding
- **Section 5** — Home Screen
- **Section 6** — Level Select (World Map)
- **Section 7** — Game Screen (Core Gameplay)
- **Section 8** — Win Dialog & Replay Mode
- **Section 9** — Settings Screen
- **Section 10** — Skin Shop & Arrow Skin Editor
- **Section 11** — Achievements & Leaderboard
- **Section 12** — Monetization & Ads
- **Section 13** — Notifications & Play Console Integration
- **Section 14** — Game Logic, Levels & AI Hint System
- **Section 15** — Final Integration, Testing & Play Console Setup
- **Section 16** — Assets Reference

## Stack

- Flutter 3.x / Dart 3.x
- Riverpod + Bloc
- GoRouter
- Hive + SharedPreferences
- AudioPlayers
- Lottie
- Google Mobile Ads
- In-App Purchase
- Google Play Games Services
